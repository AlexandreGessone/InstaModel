# InstaModel
Simulação de algumas funcionalidades presentes no Instagram

# Sobre
Neste projeto fiz implementações das seguintes funcionalidades:
- troca de cor do ícone coração quando clicado; 
- contagem do número de likes assim que o ícone for clicado;
- criação de um ícone para salvar o post. Ao ser clicado, ele fica verde indicando que o post foi salvo;
- criação de um ícone para compartilhar o post. Ao ser clicado ele apresenta uma nova janela com os ícones do Instagram, Facebook e WhatsApp. Ao clicar
sobre os ícones um alerta é exibido informando a rede social e a mensagem compartilhada. Depois disso, o campo de input é limpo;

# Stack
Utilizei o React como stack principal trabalhando os conceitos de módulos e estados dos componentes.
